# Support Bot

Base Bot WhatsApp


## Run Locally

Clone the project

```bash
  git clone https://github.com/uw-prabath/Support-Template
```

Go to the project directory

```bash
  cd Support-Template
```

Install dependencies

```bash
  npm install
```

Install dependencies **recommended**
```bash
 npm install --force
```

Start the server

```bash
  npm start
```
